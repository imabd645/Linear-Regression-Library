# LinearRegressionLib

A simple **C++ Linear Regression library** for fitting a straight line to data, predicting values, and evaluating model performance. Ideal for beginners and small projects needing a lightweight linear regression implementation.

---

## Features

- Fit a linear regression model to data
- Predict new values
- Calculate **RÂ²** (coefficient of determination)
- Calculate **MSE** (Mean Squared Error)
- Save model parameters to a file
